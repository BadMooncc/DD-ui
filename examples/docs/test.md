## 时间格式化过滤器

:::demo

```html

<div>
  <button>dasdas</button>

```

### Attributes
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
|  type  | 可选 |  'YYYY-MM-DD'/'YYYY-MM'/'hh:mm:ss'  | none | 'YYYY-MM-DD hh:mm:ss' |
